# Raycaster (GLFW Starter)

## Dependencies
### Arch / CachyOS
Run:
- `./scripts/deps_arch.sh`

### Ubuntu / Debian
Run:
- `./scripts/deps_ubuntu.sh`

## Build (preset-based)
- Debug:
  - Configure: `./scripts/configure.sh debug`
  - Build: `./scripts/build.sh debug`
  - Run: `./scripts/run.sh debug`
- Release:
  - Build: `./scripts/build.sh release`
  - Run: `./scripts/run.sh release`
